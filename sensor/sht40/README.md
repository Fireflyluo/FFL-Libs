# SHT40 温湿度传感器驱动说明文档

## 1. 概述

SHT40是一款高精度数字温湿度传感器，采用I2C接口通信。该传感器具有出色的长期稳定性、快速响应时间和低功耗特性，适用于环境监测、智能家居、工业控制等应用场景。

### 1.1 主要特性

- **温度测量范围**：-40°C 至 +125°C
- **温度精度**：±0.1°C（典型值），±0.3°C（最大值）
- **湿度测量范围**：0% 至 100% RH
- **湿度精度**：±1.8% RH（典型值），±3.0% RH（最大值）
- **低功耗**：工作电流典型值为0.4mA，睡眠模式电流<0.1μA
- **通信接口**：I2C接口（标准模式和快速模式）
- **多种测量模式**：支持不同精度和功耗的测量模式
- **内置加热器**：可选加热功能用于湿度传感器自清洁

### 1.2 驱动架构

```
SHT40 Driver
├── HAL Layer (sht40_hal.c/h)    # 硬件抽象层实现
└── Examples                      # 使用示例（参考demo目录）
```

## 2. 配置选项

### 2.1 测量模式配置

SHT40支持多种测量模式，通过命令选择不同的精度和功耗组合：

| 命令 | 温度精度 | 湿度精度 | 测量时间 | 典型功耗 |
|------|----------|----------|----------|----------|
| HIGH_PRECISION | ±0.1°C | ±1.8% RH | 8.2ms | 0.4mA |
| MEDIUM_PRECISION | ±0.2°C | ±2.5% RH | 4.0ms | 0.4mA |
| LOW_PRECISION | ±0.3°C | ±3.0% RH | 2.0ms | 0.4mA |

### 2.2 加热器模式

SHT40内置加热器，可用于传感器自清洁或特殊应用：

| 加热模式 | 功率 | 持续时间 | 电流消耗 |
|----------|------|----------|----------|
| HEATER_HIGH | 200mW | 1s | 110mA |
| HEATER_MEDIUM | 110mW | 1s | 60mA |
| HEATER_LOW | 20mW | 1s | 12mA |

### 2.3 基本使用流程

```c
#include "sht40_hal.h"

// 1. 初始化SHT40（通常只需确保I2C正常工作）
sht40_init();

// 2. 执行温湿度测量
float temperature, humidity;
if (sht40_measure(&temperature, &humidity, SHT40_HIGH_PRECISION) == SHT40_OK) {
    // 处理温湿度数据
    printf("Temperature: %.2f°C, Humidity: %.2f%% RH\n", temperature, humidity);
}

// 3. （可选）使用加热器功能
sht40_heater(SHT40_HEATER_HIGH); // 启用高功率加热1秒
```

## 3. 平台适配

### 3.1 I2C接口要求

需要实现以下I2C操作函数：

```c
// I2C写操作
int8_t sht40_i2c_write(const uint8_t *data, uint16_t len);

// I2C读操作
int8_t sht40_i2c_read(uint8_t *data, uint16_t len);
```

### 3.2 延时函数

需要提供毫秒级延时函数：

```c
void sht40_delay_ms(uint16_t ms);
```

## 4. API参考

### 4.1 初始化函数

- `sht40_init()` - 初始化SHT40传感器（主要是I2C初始化）

### 4.2 测量函数

- `sht40_measure()` - 执行温湿度测量
- `sht40_measure_temperature()` - 仅测量温度
- `sht40_measure_humidity()` - 仅测量湿度

### 4.3 加热器函数

- `sht40_heater()` - 控制内置加热器

### 4.4 状态函数

- `sht40_soft_reset()` - 软件复位传感器
- `sht40_read_serial()` - 读取传感器序列号

## 6. 注意事项

1. **I2C地址**：SHT40的固定I2C地址为0x44
2. **电源要求**：工作电压范围为2.4V至5.5V
3. **测量间隔**：建议两次测量之间至少间隔2ms
4. **环境影响**：避免将传感器直接暴露在强气流或直射阳光下
5. **PCB布局**：传感器应远离发热元件，以获得准确的温度读数
6. **校准**：虽然SHT40出厂已校准，但在关键应用中建议进行现场校准
7. **加热器使用**：加热器会产生热量，影响温度读数，使用后需等待传感器冷却
